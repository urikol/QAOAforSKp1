Assumption hker in the theorem trace_U_B_dagger_mul_U_B_rho_s_eq_one


Code assumption (Lean):


hker :

  ∀ x y : BasisIdx n,

    (∑ z : BasisIdx n, star (mixerEntry n β z x) * mixerEntry n β z y)

      = if x = y then 1 else 0


What each symbol means


1) BasisIdx n

- This is the computational-basis index type for (n+1)-qubit basis states.

- Its size is 2^(n+1).


2) mixerEntry n β x y

- This is the kernel (matrix element) K_{xy} of the mixer operator U_B.

- In matrix language, K is the matrix of U_B in the computational basis.


3) star (mixerEntry ...)

- In this context, star is complex conjugation.

- So star (K_{zx}) means overline(K_{zx}).


Formula version of hker


If we write K_{xy} := mixerEntry n β x y, then hker says:


  Σ_z overline(K_{z x}) K_{z y} = δ_{x y}


where δ_{x y} is the Kronecker delta:

- δ_{x y} = 1 if x = y

- δ_{x y} = 0 if x ≠ y


Linear algebra meaning


- The quantity Σ_z overline(K_{z x}) K_{z y} is the inner product between column x and column y of K.

- Therefore hker states that the columns of K are orthonormal.

- Equivalently: K†K = I.

- This is exactly the (left) unitarity condition for the mixer kernel.


How it is used in the trace theorem


The theorem proved in code is:


  trace_U_B_dagger_mul_U_B_rho_s_eq_one


and concludes:


  Tr((U_B†U_B) ρ_s) = 1


under hker.


The proof steps are:


1) Define the matrix of U_B†U_B by entries:


   M_{xy} = Σ_z overline(K_{z x}) K_{z y}


2) By hker, M_{xy} = δ_{xy}, so M = I.


3) Therefore:


   Tr((U_B†U_B) ρ_s)

   = Tr(M ρ_s)

   = Tr(I ρ_s)

   = Tr(ρ_s)


4) From the previously proved normalization of |s>, the density matrix ρ_s = |s><s| has trace 1:


   Tr(ρ_s) = 1.


Hence:


   Tr((U_B†U_B) ρ_s) = 1.


Important clarification


- hker is an assumption about the mixer kernel (i.e., about U_B), not about |s> itself.

- |s> contributes through ρ_s and its normalization Tr(ρ_s)=1.
